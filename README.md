## Wallet UI
A modern, clean, and smooth **Wallet User Interface** built with pure HTML and CSS.

![Wallet UI Preview](https://via.placeholder.com/800x500?text=Wallet+UI+Preview)

## Features

- Elegant and minimal wallet dashboard
- Clean typography and spacing
- Smooth animations for card selection and cash addition
- Interactive payment mode with saved cards
- Quick cash amount selection
- Fully responsive design
